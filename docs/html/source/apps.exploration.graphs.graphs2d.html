EDA_miner.apps.exploration.graphs.graphs2d module¶

This module contains the implementations of graphing functions. Every graph has its own line in the config which indicates if the graph needs both X and Y variables and whether the Y variables should be multiple.

You can freely write code in this module, without worries.
