EDA_miner.reportapp module¶

This module is meant to be used as a secondary server only for printing PDFs made inside the PDF_Layout tab.

Dash callbacks:
  • load_stuff: On button click, load the layout exported from the main app.
Notes to others:
Improvements can definitely be made regarding handling of PDF printing, with particular need of javascript functionality. Help is most welcome here, and should be prioritized.
load_stuff(_)¶

Load the layout exported from the main app when click.

Parameters:_ (int) – number of times the button was clicked, not important.
Returns:loaded from bytes, the dash components making up the PDF report.
Return type:pdf_layout (obj)
